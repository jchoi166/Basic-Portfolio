Bookeeping App

[Group Project]


A standard accounting app that lets you keep track of expenses and revenue through journal entries and run monthly/quarterly/yearly reports. The app was made entirely with React and Material UI (React CSS Library) with each person assigned to a different component. I was in charge of the “search journal entries” component where the user can search for journal entries using a journal ID or Approver/Preparer name. The search function has an autocomplete feature and the search results are displayed via cards for each journal entry. We used Axios to make AJAX calls and MongoDB as our database with Mongoose to write out our queries.
